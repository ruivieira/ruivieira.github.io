Installing

Installing pweave is a matter of simply running1:

pip3 install pweave

At the moment of writing, the editor which, IMO, has the best support for pweave is Atom (using Hydrogen). To install the necessary packages run:

apm install language-weave Hydrogen
apm install language-markdown atom-html-preview pdf-view

  1. I recommend using a separate pyenv for this. ↩︎
